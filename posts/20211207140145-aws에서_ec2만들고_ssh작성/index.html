aws에서 ec2만들고 ssh작성

AWS에 로긴한다.

ec2를 만든다.

  • ec2를 만드는 것은 그대로 next, next해서 진행한다.
  • key를 만드는 부분이 나올때, key를 생성한다. 그리고 그 key .pem으로 끝나는 그키..를 다운받아서, local의 .ssh/아래 붙여넣는다.

ec2의 주소를 찾는다.

  • instance에 보면 public ip4 주소가 있다.

접속해 보자.

  • 접속방법은 다음과 같다. 
    ssh -i ./ssh/<인증서.pem> ubuntu@ip주소
  • ec2의 instance들은 처음에 ubuntu 사용자가 있다. 그래서 ubuntu로 접속한다.

root 비밀번호를 설정

  • sudo 작업을 해야 하기 때문에 
    sudo passwd root
