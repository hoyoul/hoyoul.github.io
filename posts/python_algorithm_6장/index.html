유효한 palindrom

주어진 문자열이 palindrom인지 확인하라. 대소문자를 구분하지 않으며 영문자와 숫자만을 대상으로 한다.

def palindrom(substr):

  if len(substr) == 1 or len(substr) == 0:
    print("true")
    return True
  else:
    start = substr[0]
    end = substr[len(substr)-1]
    if (start == end):
      print(substr)
      print(substr[1:len(substr)-1])
      palindrom(substr[1:len(substr)-1])
    else:
      print("false")
      return False

print(palindrom("abcdcba"))
