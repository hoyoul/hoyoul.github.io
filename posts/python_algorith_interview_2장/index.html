generator

개요

generator의 형태는 함수 형태다. return할 값을 yield로 나타낸다. generator를 호출하지 않고 next(gen)로 호출하며, 호출 될 때마다 yield값이 return된다. 다음 예제를 보자.

example1

def generator():
  yield 1
  yield 2
  yield 3

g = generator()
print(next(g))
print(next(g))

example2

def get_natural_number():
    n=0
    while True:
	n += 1
	yield n

g = get_natural_number()
print(next(g))
print(next(g))
print(next(g))

range

for-loop에서 주로 쓰인다. generator에서 return될 값들을 yield로 나타내듯, range도 내부적으로 return될 값들을 만들어낸다.

for i in range(4):
  print(i)

enumerate

enumerate객체는 list,set,tuple값을 인자로 받아 만들어진다. 만들어진 객체를 사용하려면 range와 같이 for-loop와 연동해서 사용한다. enumerate객체는 collection의 item의 index를 tuple로 묶어주는 역할을 한다. 독자적으로는 별로 사용되지 않는듯하다.

a = [1,2,3]
e = enumerate(a)

for i,v in e:
    print(i,v)
