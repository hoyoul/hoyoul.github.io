tkinter

  • Tcl/Tk interface를 말한다.
  • Tcl은 Tool Command Language의 약자다.
  • Tk는 tool kit의 약자다.
  • Tcl/Tk는 GUI programing이 가능하게 window와 widget을 제공한다.

GUI programming

  • root window: GUI program은 윈도우가 반드시 하나 있어야 한다. root window는 tk()로 만들어진다.
  • widget
    • simple widget
      1. label
    • container widget
      1. frame : window도 widget을 담을 수 있지만, widget을 담는 widget이 필오하다. Frame이라는 container다.
