실습게시판 list 만들기

table 만들기

list.html이란 파일을 만든다.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <title>Document</title>
  </head>
  <body>
  </body>
</html>

head에는 css,js같은 내용이 들어가고, body에 page에 관한 내용이 들어간다. browser에서 보기위해서 emacs에서는 M-x imp-visit-file을 실행시키면 된다.

table tag를 사용해서 게시판을 만든다. emmet을 사용해서 다음과 같이 입력한다.

table>thead>tr>td

이렇게 만들면 thead 아래 tr(table row)와 td(table data)를 넣겠다는건데, 즉 제목을 표시한다. html코드를 보면 다음과 같다.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <title>Document</title>
  </head>
  <body>
    <table>
      <thead>
	<tr>
	  <td>제목</td>
	</tr>
      </thead>
    </table>
  </body>
</html>

mdn 참고

table에 대한 정보를 알고 싶다고 한다면 mdn사이트를 추천한다. web에관한 좋은 문서들이 있다고 한다. 여기서 주로 검색한다.

table 추가작업

table table head엔 #,제목, 작성자, 일시를 입력하고, body에는 3개의 줄을 입력한다. emmet을 사용해서 간단히 할 수 있다.

table>thead>tr>td*4^^tbody>tr*3>td*4

css의 사용

tag에 style넣기

css를 tag에 직접 적용하는 방식이 있다. tag에 style이란 속성을 넣는 것이다. 이것으로 table header에 적용해 보자.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <title>Document</title>
  </head>
  <body>
    <table style="width:500px;">
      <thead>
	<tr>
	  <td>#</td>
	  <td>제목</td>
	  <td>작성자</td>
	  <td>일시</td>
	</tr>
      </thead>
      <tbody>
	<tr>
	  <td>3</td>
	  <td>게시글입니다</td>
	  <td>이태화</td>
	  <td>2022.04.25</td>
	</tr>
	<tr>
	  <td>2</td>
	  <td>게시글입니다</td>
	  <td>이태화</td>
	  <td>2022.04.25</td>
	</tr>
	<tr>
	  <td>1</td>
	  <td>게시글입니다</td>
	  <td>이태화</td>
	  <td>2022.04.25</td>
	</tr>
      </tbody>
    </table>
  </body>
</html>

tag에 style이란 속성을 넣을때는 symbol:값의 형태다. style은 symbol이고 width는 속성값이다.

head에 style tag로 넣기
