emacs_library_or_custom el test하는 법

개요

emacs에서 library를 가져다 쓰거나, 새로운 el파일을 만들어서 사용할 때, 기존의 설정파일때문에 문제가 될 수 있다. 기존의 설정을 사용하지 않고 emacs를 시작하는 법이다.

test

간단한 el파일을 만든다.

;; Load the publishing system
(require 'ox-publish)

(message "Build complete!")

emacs로 테스트한다.

emacs -Q --script build-site.el

  • -Q를 주는것은 기존의 설정을 사용하지 않겠다는 의미다.

  • –script는 해당 script를 실행하겠다는 뜻이다.

    예를 들어서, emacs -Q build-site.el을 하면, 기존 설정을 사용하지 않고, gui emacs를 실행한후 해당 파일을 open한다. emacs –script build-site.el을 하면, 기존설정을 가진 emacs가 script를 실행한다.
