기본 명령어

(C-x g)magit status

  • 가장 많이 쓰이는 기능 interface를 보여준다. 만일 현재 buffer가 git으로 관리하고 있다면, git status를 보여준다. git으로 관리되고 있지 않다면, git으로 관리되는 폴더를 찾는다.
