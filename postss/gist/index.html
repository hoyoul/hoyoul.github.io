emacs의 설정에 문제가 생겼거나 새로운 설정으로 할때

  • elpa를 지운다.
  • init.el을 지운다.
  • emacs를 실행 시킨다.
  • emacs.org를 buffer에서 연다.
  • tangle을 evaluate한다. => emacs.org가 init.el을 만든다.
  • 다시 emacs를 시작하면, init.el에서 package들을 elpa에 download한다.
  • 내가 작성한 파일이거나, package manager에서 관리하지 않는 파일들은 lisp폴더에 저장하고 add-to-list로 load-path를 걸어둔다는 것을 잊지 말자.
