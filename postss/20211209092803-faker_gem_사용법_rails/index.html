faker gem 사용법 [rails]

faker: dummy data넣어주는 gem

(1) rails g model table_name column1:type column2:type으로 model생성

1) rails g model Pressure dong:integer roomtype:string pressure:float temperature:float gettingtime:datetime
2) rails g model Watermeter dong:integer roomtype:string flowrate:float totalvolume:float gettingtime:datetime

(2) rails db:migrate

(3) gemfile에 gem ‘faker’ 추가

gem 'faker', :git => 'https://github.com/faker-ruby/faker.git', :branch => 'master'

(4) bundle install

(5) db/seed.rb 수정

  • seed는 db에 초기 데이터를 채워주는 역할을 한다. seed money 생각하면 된다.
  • require faker는 faker의 random한 data를 넣어주겠다고 생각하고 faker를 load한다고 보면 된다.
  • watermeter라는 모델에 해당 field를 faker에서 동적으로 만들어주는 data를 assign하면 된다.
  • faker github에 가면 비슷한 field를 찾을 수 있다. 이것을 넣어주면 된다. 
    
100.times do
   Watermeter.create(
   dong: Faker::Number.number(digits: 3),
   roomtype: Faker::Lorem.characters(number: 1),
   flowrate: Faker::Number.decimal(l_digits: 1, r_digits: 4),
   totalvolume: Faker::Number.number(digits: 6),
   gettingtime: Faker::Date.between(from: '2021-10-23', to: '2021-12-25')
   )
end

(6) rails db:seed

에러가 나면, rails db:drop db:create db:migrate db:seed를 해준다.
