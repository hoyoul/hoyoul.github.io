[emacs]cl-lib 사용법

common lisp package

emacs의 거의모든 program들은 common lisp package를 사용한다. 그중 cl-lib을 많이 사용한다.따라서 code를 분석하거나 사용할때 cl-lib를 사용할 줄 알아야 한다.

참고:https://www.gnu.org/software/emacs/manual/html_mono/cl.html

usage

(require ‘cl-lib)

위와 같이 load해서 사용한다.
