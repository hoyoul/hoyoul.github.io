co-worker의 입장

  • host가 git repository를 만들고, co-worker에게 초대장을 보냈다고 가정한다.
  • git clone으로 repo를 가져온다.
  • branch를 만든다. git branch brchA or git branch hoyoul
  • git checkout brchA로 branch로 옮긴 후, commit하고 작업을 하면 된다.
  • 작업이 끝났으면, git checkout main한 후, git merge brchA로 merge한 후에 git push origin master를 하면 된다.
