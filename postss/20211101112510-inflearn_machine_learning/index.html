1.1 python and pip vs anaconda

  • package 설치 프로그램 (영상에선 anaconda 추천)
  • 내 경우, pyenv+ virtualenv를 사용할 예정
  • pyenv(python interpreter 설치, 설정)
  • pyenv-virtualenv(가상환경의 설치,설정; 가상환경-package가 설치되는 공간)
brew install pyenv
brew install pyenv-virtualenv
  • .zshrc추가
# pyenv command
eval "$(pyenv init -)"

# pyenv-virtualenv command
eval "$(pyenv virtualenv-init -)
  • python 설치할 버전 확인
pyenv install --list
pyenv install --list | grep <version>
  • python 설치
pyenv install <version>
  • python 설정
pyenv global version
  • 설치된 python version 확인
pyenv versions
  • 가상환경 만들기

가상환경을 만들고 pyenv versions에서 가상환경을 확인할 수 있다. pyenv는 특정 python version의 환경을 만들 뿐이다. 동일한 python interpreter를 사용하지만, package만 다르게 설치하기 위해서는 pyenv-virtualenv를 사용해야 한다.

pyenv virtualenv <version> <name>
